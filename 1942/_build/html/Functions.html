Functions¶


main
1942.main()¶

Main function responsible for calling functions and Initial declarations of all varaiables, and arrays

Variables

  • alt – Used to change x direction of small silver plane

  • run – Tells when a small green ship has a missile alive

  • srl, mrl, lrl (r,) – Lists for the rocket lists of enemy planes and player

  • silverList, mediumList, largeList (greenList,) – Lists for enemy planes

  • level – Tells what level the player is on

  • hits, misses, fires, destroyed (totalScore,) – For keeping stats on players shooting

  • dead – Tells if the player is dead or alive on level end

  • enemyRocket, player, greenS, silverS, greenM, greenL, logo (rocketImg,) – For sprite images

  • bg – Background images list

  • counter – Tells how long the game level has gone on

  • itr – Keeps track of number of player rockets

  • PAUSE – Play/pause music of class Pause

  • gameOn – Bool to tell if the main game loop runs
